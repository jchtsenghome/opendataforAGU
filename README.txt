How to use codes

I. IPV ISOMAP

1) make -f compilecalipv5disomap
To generate calipv5disomap.exe

2) eidt calipv5disomap.nmlst
Arrange the input data directory. The input file is in ./data directory

II. IPV contour length

1) gfortran getcontlengthv5.f -o getcontlengthv5.exe

2) exexute getcontlengthv5.exe
To generate the data that are used for plotting IPV contour length.
